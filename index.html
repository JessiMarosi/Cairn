Cairn

Local-first developer infrastructure software for secure, auditable, AI-assisted development.

What is Cairn?

Cairn is a cross-platform developer workbench designed to keep projects private, auditable, and fully under user control. It emphasizes local-first operation, explicit project boundaries, and security-aware workflows.

Core Principles

  • Local-first and offline-capable
  • No required cloud accounts
  • Explicit on-disk projects
  • Security and auditability by design
  • Clear separation between UI and Core

Status

Cairn is in early development. Schemas, security boundaries, and architecture are being finalized prior to implementing core functionality.
